# 🚕 TaxiGo - Telegram Mini App

تطبيق تاكسي متكامل داخل تيليجرام مع دعم العربية وتتبع مباشر.

## الإعداد السريع
1. انسخ ملفات الترحيل إلى Supabase SQL Editor.
2. أنشئ `.env.local` في `frontend` بمتغيرات Supabase.
3. `npm install` في `frontend` و`bot`.
4. انشر البوت باستخدام `wrangler deploy`.
5. انشر الواجهة على Vercel.

راجع التعليمات الكاملة في ملف README الشامل.
